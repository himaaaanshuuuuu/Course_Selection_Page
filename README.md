# Course Selection Portal

A responsive and interactive Course Selection Portal built using HTML, CSS, and JavaScript. The project allows students to browse available courses and select a course dynamically through an intuitive user interface.

---

## Features

- Responsive layout for all screen sizes
- Interactive course selection using JavaScript
- Modern UI design with hover effects
- Dynamic display of selected course
- Clean and organized code structure

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
